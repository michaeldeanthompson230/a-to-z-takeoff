# Estimating Rules

## Exterior Walls

* Read wall type from plans when available
* Detect wall thickness when legend is unavailable
* Default stud spacing per plan
* Default 16" O.C.
* Double top plate
* Single bottom plate
* 10% framing waste
* 10% sheathing waste

## Interior Walls

* Thin walls = 2x4
* Thick walls = 2x6
* Wet walls default to 2x6
* Double top plate
* Single bottom plate
* 10% framing waste

## Headers

Priority:

1. Plan callout
2. Beam schedule
3. LVL schedule
4. Default double 2x12 #2 Southern Yellow Pine

Flag for review:

* Openings greater than 6'
* Garage doors
* Multi-panel sliders
* Structural openings

## Engineered Lumber

Only include when called out on plans.

Examples:

* LVL
* I-joist
* Floor truss
* Roof truss
* Rim board
* Engineered beams

Never guess engineered lumber.

## Sheathing

Default wall sheathing:

* 7/16 OSB

Available options:

* CDX
* OSB
* ZIP System

## Housewrap

Default:

* 9' x 150' roll

## Fasteners

Framing and sheathing:

* 2½" .131 electrogalvanized ring-shank coil framing nails

All other fasteners:

* Standard construction fasteners

## Waste Factors

Framing:

* 10%

Sheathing:

* 10%

Additional waste factors may be added later by package type.
