# a-to-z-takeoff
A-Z Takeoff

A-Z Takeoff is an AI-assisted construction takeoff and estimating platform designed to convert residential construction plans into complete material packages and quote-ready bills of material.

## Mission

Automatically convert residential plan PDFs into lumber-yard-quality material packages with minimal estimator input.

## Long-Term Goal

A user uploads a permit set PDF and receives:

* Framing package
* Siding package
* Drywall package
* Trim package
* Door package
* Window package
* Cabinet package
* Countertop package
* Roofing package

along with customer-ready quote output.

## Technology Stack

* Python
* PySide6
* OpenCV
* PyMuPDF
* SQLite
* Pandas
